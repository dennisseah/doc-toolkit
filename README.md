# doc-toolkit

document toolkit

## Setup

1. install Python 3.10
2. install Poetry
3. in git root folder
   1. python -m venv .venv
   2. poetry shell
   3. poetry install

## IDE

1. vscode
   1. Cmd-Shift-P type Show Recommended Extension. Install them
   2. open a terminal and type pre-commit install
